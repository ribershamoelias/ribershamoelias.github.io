# Riber Shamo Elias - Persönliche Website

> Portfolio, Kontakt & Linkseite

**Live:** https://ribershamoelias.com

Dieses Repository enthält deine statische persönliche Website und eine separate Linkseite.

- `index.html`/`style.css`/`script.js` – Hauptseite mit Hero, About, Projekte, Skills, Kontakt
- `my-links/` – eigenständige Seite für alle deine Social-Links (bearbeite `links.json`)

## Schnellstart

1. Bearbeite `index.html` (Name, Foto, Texte) und `my-links/links.json` (deine Links).
2. Teste lokal durch Öffnen der HTML-Dateien oder mittels `python3 -m http.server`.
3. Erstelle ein GitHub-Repository `dein-username.github.io` und pushe den Code.
4. Aktiviere GitHub Pages in den Repository‑Einstellungen. Deine Seite ist in wenigen Minuten live.

Weitere Anpassungen findest du in [PERSONAL_WEBSITE.md](./PERSONAL_WEBSITE.md).

Viel Erfolg! 🖤
