# 👤 Persönliche Website - Riber Shamo Elias

Einfach anpassbare statische Seite mit Portfolio, Projekten und Kontakten.

## 🎯 Features

- Hero-Sektion mit Bild und Motto
- Über mich, Projekte, Skills, Kontakt
- Responsive Design (Desktop/Mobile/Tablet)
- Dunkles Theme mit Lila-Akzenten
- Smooth Scroll Navigation
- Leichtgewichtig: nur HTML/CSS/JS

## 📂 Dateien

```
/
├── index.html          # Hauptseite
├── style.css           # Styling
├── script.js           # kleine Interaktivität
└── my-links/           # Linkseite für Social-Profile
```

## ⚙️ Anpassungen

- **Foto**: ersetze `src` im `<img class="avatar">` in `index.html`
- **Name / Texte**: bearbeite Überschriften und Absätze direkt in `index.html`
- **Farben**: ändere Hexwerte in `style.css` (z.B. `#667eea`, `#764ba2`)
- **Links**: bearbeite `my-links/links.json` mit deinen URLs
- **E‑Mail**: `mailto:`-Link im Kontaktbereich aktualisieren

## 📱 Mobile Tests

```bash
python3 -m http.server
# besuche http://localhost:8000
```

## 🚀 Deployment

1. Repository anlegen: `dein-username.github.io`
2. Alle Dateien pushen (siehe README.md)
3. GitHub Pages aktivieren (Settings → Pages)
4. Optional: eigene Domain eintragen
