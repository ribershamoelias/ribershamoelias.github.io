FAQ

Common questions.

Is MineSkin free?

Yes — completely free, forever. No account, no paywall, no ads.

Do I need to install anything?

No. MineSkin runs entirely in your browser. Just open the editor and start drawing.

Will my work be saved?

Yes. Your skin is auto-saved to your browser's localStorage every time you draw. It will be there when you come back.

What file format does it export?

PNG — the only format Minecraft accepts. The file is named yourprojectname_mineskin.png.

Can I import an existing skin?

Yes. Drag and drop any 64×64 or 128×128 PNG onto the editor, or use the Import button.

Does it work on mobile?

MineSkin is optimized for desktop and tablet. Smartphones are technically supported but the experience is limited due to screen size.
